# Instalación

Añadir repositories del archivo composer.json el siguiente codigo:

```
"repositories":[
    ...    
    {
        "type": "vcs",
        "url": "https://github.com/gragot/laravel-log.git"
    }
    ...
],
```

Ejecutar composer require para descargar el repositorio:

```
composer require gragot/laravel-log
```
